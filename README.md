# Atelier 33 Barber — concept AeroCode

Site vitrine fictif de barber shop créé avec React et Vite.

## Démarrage local

```powershell
npm install
npm run dev
```

Ouvrir l'adresse affichée par Vite, généralement `http://localhost:5173/`.

## Build de vérification

```powershell
npm run build
npm run preview
```

## Personnalisation rapide

- Textes et données : `src/App.jsx`
- Apparence et responsive : `src/styles.css`
- Titre et description SEO : `index.html`
- Illustrations : dossier `public`

Le commerce, l'adresse, les coordonnées et les avis sont fictifs. Ne pas présenter ce projet comme un vrai client.
